## ðŸŽ® Game Rules

### 1) Placement phase
- Each player places **3 pieces** on the board (X or O)

### 2) Movement phase
- If nobody wins after placing the pieces, players start **moving** their pieces on the board

### 3) Victory
- The winner is the player who manages to **align 3 pieces**
